Fraud Detection in Financial Transactions

  • GitHub:   
  • Project:   

Fraudulent activities in financial transactions pose significant challenges to maintaining trust and security in modern banking systems. This project addresses the critical issue of fraud detection in financial transactions through the application of big data analytics and machine learning techniques.

Using a synthetic dataset, "Synthetic Financial Datasets For Fraud Detection", obtained from Kaggle, our research contributes to the domain of fraud detection within the emerging field of mobile money transactions. Through exploratory data analysis and feature engineering, we uncovered hidden anomalies indicative of fraudulent behavior within the dataset. Using machine learning algorithms such as logistic regression classifier, Random Forest Classifier, LGBM Classifier, and XGBClassifier, we analyzed transaction data to identify potential fraud patterns with high accuracy, precision, recall, F1 score, and area under the curve.

Additionally, we created an interactive dashboard using Tableau to present our findings in a user-friendly and comprehensible manner. Notably, our models achieved an accuracy of over 95% in detecting fraudulent transactions, outperforming existing methods. These findings provide insights for financial institutions seeking to improve fraud detection systems and protect their customer's assets. By contributing to the advancement of fraud detection systems, this research sets new standards for securing financial transactions in the digital age.
